# SlashBlade Patchouli

A lightweight mod that adds integration between **SlashBlade: Resharped** and **Patchouli**.

## Introduction

This mod adds **Patchouli** support for **SlashBlade: Resharped**, allowing players to view SlashBlade-related content
through the Patchouli guidebook.

You can craft the SlashBlade handbook by combining a **Blade Soul Trapezohedron** with a book.
